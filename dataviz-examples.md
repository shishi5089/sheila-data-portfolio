# Data visualization examples
### OECD Government Debt Highlight Table

This visualization shows government debt as a percentage of GDP across OECD countries over time. The highlight table allows for quick comparison of debt levels between countries and across years, with color indicating relative magnitude. Values above 100% are visually emphasized through the diverging color scale.

**Source:** OECD Data – General government debt (% of GDP)
